# Email Header Parser & Validator

A web application for parsing and validating email headers.

## Features
- Parse email headers
- Validate email addresses
- Check date formats
- Identify authentication headers (SPF, DKIM, DMARC)
- Visual feedback for valid, invalid, and missing headers

## Development
\`\`\`bash
npm install
npm run dev
\`\`\`

## Build
